# DEPI DevSecOps Project Showcase

A React + Vite visual documentation portal for the DEPI DevSecOps Project — MIND Notes App.

## Local Run

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Vercel Settings

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

## Security Notice

Do not publish private admin passwords, tokens, SSH keys, cloud credentials, or `.pem` files in this repository.
